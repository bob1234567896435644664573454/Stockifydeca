# Code Review Skill

## Purpose
To audit the codebase for bugs, security risks, UI/UX inconsistencies, and architectural flaws.

## Capabilities
- Static analysis (linting, type checking)
- Pattern matching for known issues
- Architecture review
- Security scanning (auth guards, RLS)

## Usage
Run `/code_review` to initiate a full repo sweep.
