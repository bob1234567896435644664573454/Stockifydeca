# UI/UX Pro Max Skill

## Purpose
To elevate the application's design to a competition-grade level, focusing on aesthetics, usability, and responsiveness.

## Capabilities
- Design system implementation (typography, color, spacing)
- Component creation (shadcn/ui, custom)
- Micro-interactions and animations
- Accessibilty auditing

## Usage
Run `/enhance_ui` or `/ui_ux_pro_max` to apply design improvements.
